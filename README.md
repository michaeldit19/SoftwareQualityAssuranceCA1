# SoftwareQualityAssuranceCA1
This repository will link to the handbook detailing and describing the coding conventions outlined in said handbook.

This repository will focus on three aspects of Quality Assurance within the workplace and within a development team. These aspects are 
Coding Standards, Unit Testing and Version Control containing Merging and Branching. By adhering to these to these qualities, we aim to achieve:

1. Universal understanding of the code,
2. Greater efficiencey within the development team,
3. Concise and formatted code,
4. Ease of merging and prodcing high quity code and 
5. Better gauage of where tickets are at in development
